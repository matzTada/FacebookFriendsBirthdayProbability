## Problem
だんだん数学の問題にしていきます．  
Facebookで何人友達がいたら，毎日誕生日通知が来るのか  
→N人友達がいるときに，365日すべてにおいて少なくとも1人の友人の誕生日がある  
→N個の箱に対して重複を許して365個の誕生日   
実はFacebookは関係ありませんな．
